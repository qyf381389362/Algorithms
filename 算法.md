### 记录常见的算法

使用的编程语言是java

---

#### 判断一个数是否是素数：

```java
public static boolean isPrime(int N){
  if (N<2) return false;
  for(int i=2;i*i<=N;i++){
    if(N%i==0) return false;
  }
  return true;
}
```

#### 计算调和级数

调和级数：1+1/2+1/3+1/4+...+1/N

```java
public static double H(int N){
  double sum = 0.0;
  for(int i=1;i<=N;i++){
    sum+=1.0/i;
  }
  return sum;
}
```
